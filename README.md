## Features
	'/time %hh:mm%' 
	Get the possible outcome of production.

	'/isgood %name%' 
	Get quick summary on T-doll or fairy.

	'/info %name%' 
	Get quick info on T-doll or fairy.

	'/poke' 
	Don't tease me.

	'/slap %name%' 
	Teach someone or something a lesson.

	'/help' 
	Get commands list.
